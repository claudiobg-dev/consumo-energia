# ⚡ Calculadora de Consumo de Energia
## 📌 Sobre o projeto

Este projeto é um sistema simples em Python que calcula o consumo de energia elétrica de um aparelho.

O objetivo é ajudar o usuário a entender quanto um equipamento consome no dia a dia, com base na potência e no tempo de uso.

## ⚙️ Fórmula utilizada

O cálculo do consumo de energia é feito com a seguinte fórmula:

Consumo (kWh) = (Potência (W) × Tempo (h)) ÷ 1000

## Digite as informações solicitadas:

Nome do aparelho: chuveiro elétrico

Potência (em watts): 6800

Tempo de uso (em horas): 4
